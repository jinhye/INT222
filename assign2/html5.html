HTML5

HTML5 new form attributes

The HTML input element is used to create interactive controls for web-based forms in order to accept data from the user. The semantics of an input varies considerably depending on the value of its type attribute.

How to set autofocus

<input type="text" name="input1" id="input1" autofocus >

How to set placeholder

<input type="text" name="input2" id="input2" placeholder="sample text" >

How to set required

<input type="text" name="input3" id="input3" required >

How to create datalist

<input list="browsers"> <datalist id="browsers"> <option value="Chrome"> <option value="Firefox"> <option value="Internet Explorer"> <option value="Opera"> <option value="Safari"> </datalist>

How to set email format

<input type="email" name="input4" id="input4" >

How to set url format

<input type="url" name="input5" id="input5" >
