HTML5 & CSS Normalize

HTML5 is the latest evolution of the standard that defines HTML (Hyper Text Markup Language), with new elements, attributes and behaviours.

This new version is really popular because it makes possible to do things that previously required programming languages to do, like data storage and geolocation.

The tags and functionality included in HTML are defined by the World Wide Web Consortium (W3C). HTML, like software, comes out in releases.

Normalize.css is a small file that provide better cross-browser consistency in the default style of HTML elements.

It can also be used as an alternative to CSS resets. Normalize.css preserves uselful browser defaults, improves usability and helps explain the code, using comments and detailed documentation.
